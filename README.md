# KUROI FLIP

A decentralized gambling application built on Aptos blockchain with NFT holder benefits and multiple games.

## Features

- **Coinflip Games**: Classic heads or tails gambling
- **Rocket Crash**: Cash out before the rocket crashes!
- **NFT Holder Benefits**: Special dividends and rewards for KUROI NFT holders
- **Multiple Games**: Dedicated pages for different game types
- **Modern UI**: Beautiful, responsive design with KUROI branding
- **Wallet Integration**: Seamless Aptos wallet connection

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm start`
4. Build for production: `npm run build`

## Technologies

- React 18 + TypeScript
- Tailwind CSS + Lucide React Icons
- Aptos SDK v1.10.0
- React Router DOM
